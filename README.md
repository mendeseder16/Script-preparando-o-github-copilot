# Script-preparando-o-github-copilot

Requisitos Prévios
Ter uma conta no GitHub.
Possuir uma assinatura ativa (ou período de teste) do GitHub Copilot.
Ter o Visual Studio Code instalado.

2. Preparação do Ambiente (Passo a Passo)
Instalação da Extensão:
No VS Code, abra a aba de Extensions (Ctrl+Shift+X).
Pesquise por "GitHub Copilot".
Instale também o "GitHub Copilot Chat" para ter a interface de conversa.
Autenticação:
Clique no ícone de "Sino" ou no avatar de usuário no canto inferior esquerdo.
Selecione "Sign in to GitHub".
Autorize o acesso no seu navegador.
Verificação:
Um pequeno ícone do Copilot (um robô) aparecerá na barra de status inferior. Se ele estiver lá, o ambiente está pronto.

3. Praticando com o Copilot
Tente estas três formas de interação para testar a ferramenta:
Sugestão por Comentário:
Escreva # Função para validar linha digitável de boleto e aperte Enter. O Copilot sugerirá o código. Pressione Tab para aceitar.

Sugestão em Tempo Real:
Comece a digitar def calcular_juros(valor, dias):. Ele tentará completar a lógica inteira para você.

Copilot Chat (Ctrl+I ou Ctrl+Alt+I):
Selecione um trecho de código e peça: "/explain" (para explicar o código) ou "/tests" (para gerar testes unitários automaticamente).

4. Dica de Ouro: "Prompt Engineering"
Quanto mais contexto você der, melhor será o código. Se estiver trabalhando no seu serviço de autenticação de boletos, mantenha os arquivos de classe abertos no VS Code; o Copilot "lê" as abas abertas para entender o contexto do seu projeto.
